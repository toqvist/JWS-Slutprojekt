Welcome to the API!

Valid endpoints:

Calculator

Performs simple mathematical operations on two values

/calc

GET

Parameters: operation, value1, value2

Supported operations: addition, subtraction, multiplication

Example request: /calc?operation=addition&value1=1&value2=2

Rock-Paper-Scissors

Play rock paper scissors!

/RPS or /SSP

POST

Parameters: 'rock' or 'scissors' or 'paper'

GET

No parameters

/RPS

CSV as JSON

Returns a csv file as a JSON

/readCSV

GET

No parameters

Random Image

Returns a random image

/img

GET

No parameters
